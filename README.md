# Statistical Portfolio Risk Management and Variance Modeling

This project is a desktop-based financial software developed in collaboration with my teammate, **Fatih Özdemir**. The application combines software engineering, mathematics, and finance to provide a data-driven investment decision support system. 

Through this project, we practically applied concepts such as Object-Oriented Programming (OOP), user interface design, data processing, and algorithm development.

## 🛠️ Technologies & Tools
* C#
* .NET Framework
* Windows Forms
* SQL

## 📌 Key Features
* **Secure Authentication:** User verification and secure login screen.
* **Main Dashboard:** Displays up-to-date financial data and interactive charts.
* **Stock Analysis System:** A comprehensive list of stocks with detailed technical analysis screens.
* **Algorithmic Decision Support:** Evaluates technical indicators to generate automated Buy-Sell-Hold signals.
* **Investment Tracking:** Dedicated Portfolio and Watchlist modules for users to monitor their assets.
* **Financial Metrics:** Tabular presentation of stock returns, losses, trading volume, volatility, and various technical indicators.
